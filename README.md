# Codex-ChatGPT

This project allows all users to ask ChatGPT-3 to answer their questions. No registration is needed, just start chatting!

![chat-gpt](chat-gpt.jpg)

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)

## Installation

1. Clone this repository
2. Install the required packages using `npm install`

## Usage

1. Run `node app.js`
2. Start chatting with ChatGPT-3!
